# Cinematic Language Guide  
## Generational Universe · Hopeful Educational Cinema

---

## Lens philosophy

Prefer **eye-level empathy**.

| Intent | Lens / approach |
|--------|-----------------|
| Trust | Normal ~35–50mm equiv; eye height |
| Wonder | Slightly wider; character small in sacred space |
| Precision | Longer lens compression on tools/evidence |
| Intimacy | Clean CU; eyes sharp; never grotesque pore fetish |

Forbid: found-footage shake · Dutch angles for “cool” · continuous 24mm distortion on faces as style.

Primary delivery formats include vertical 1080×1920 for shortform — compose for **face + gesture readable** in that frame. Architecture still respects Bible.

---

## Camera movement

Motivated. Soft. Breath-like.

| Allowed | Forbidden |
|---------|-----------|
| Gentle push into discovery | Whip-pan chaos |
| Soft lateral to follow teaching gesture | Orbit flex for ego |
| Locked frame for evidence clarity | Constant unsettled handheld |
| Eye-led reframes | Camera hiding missing performance |

Emotion drives whether we move; pedagogy drives when we hold.

---

## Shot composition

- Subject + evidence + air  
- Rule of thirds with soft Generational bias toward eyes  
- Leading lines as discovery arcs (table edge, window mullion)  
- Never hide the teaching hand or the sample  

Foreground plants/glass OK if they frame hope — not if they obscure learning.

---

## Depth

Layers: soft FG · sharp subject · readable mid evidence · quiet BG architecture.  
Fog only as soft atmospheric hope — never horror haze.

---

## Lighting (production)

Signature from constitution:

**Warm key · soft fill · controlled teal rim · honest reflections.**

| Mood | Key |
|------|-----|
| Curiosity | Soft cool fill + warm key balance |
| Discovery | Teal accent rise + gold kick |
| Hope | Window gold dominant |
| Compassion | Warm sand wrap |
| Failure/recovery | Lower contrast warm; then gold return |

---

## Color grading

Stay inside palette.  
Lift blacks softly — never crushed void.  
Teal retained as trust, not teal-orange blockbuster cliché wash.  
Skin/shell stay warm; do not grey them “for prestige.”

---

## Mood board law

Every episode board asks:

1. Which emotion of Curiosity / Trust / Hope / Discovery / Intelligence / Compassion / Wonder is primary?  
2. Does grade + light + lens serve that emotion?  
3. Would a paused frame pass the Generational pause test?

If no — the shot is unfinished.
