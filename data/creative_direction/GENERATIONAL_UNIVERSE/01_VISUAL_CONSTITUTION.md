# Visual Constitution  
## Generational Universe · Hopeful Educational Cinema

**Status:** Permanent law  
**Break these rules → break the brand.**

---

## 1. Visual philosophy

### Emotional contract of every frame

| Emotion | Frame must feel like… | Frame must never feel like… |
|---------|------------------------|------------------------------|
| Curiosity | An invitation to look closer | A quiz trap |
| Trust | Someone worthy of a child’s attention | A cold expert talking down |
| Hope | Tomorrow can be built | Nihilism / apocalypse chic |
| Discovery | Shared “aha” | Spectacle without understanding |
| Intelligence | Calm clarity | Jargon fog / smug genius |
| Compassion | Care for the learner | Pity or propaganda |
| Wonder | World is larger and kinder than expected | Cheap jump-scare wonder |

### Medium definition

**Hopeful Educational Cinema** = feature-appeal stylization + documentary respect for truth + theatrical care for feeling.

Not cartoon slapstick.  
Not desaturated clinic.  
Not spaceship chrome opera.

---

## 2. Shape language

### Primary geometric language

| Intent | Primary forms |
|--------|----------------|
| Trust / warmth | Soft rounds, gentle ovals, cushioned corners |
| Intelligence | Clean verticals, calm rectangles, quiet alignment |
| Discovery | Soft arcs, open curves, pathways that lead the eye |
| Brand tech | Rounded rectangles (panels, screens, pins) |

### Secondary forms

Soft triangles for roof/light wedges · gentle hex only as micro-structure in science graphics (never armor) · circles as focus marks (samples, seals), not as default robot joints.

### Negative space

Breathing room is brand.  
Crowded HUD, sticker walls, and prop clutter break Generational.  
Every composition should have a **quiet zone** where the eye rests — often dawn-white wall, sky, or table negative space.

### Silhouette philosophy

Recognizable in blackout.  
Character and key props must read without color.  
Environments signal place-type in massing alone (tall windows = lab/school; soft domestic furniture block = home).

### Hard vs soft

| Soft (default for living beings & furniture) | Hard (controlled for tools & truth) |
|----------------------------------------------|-------------------------------------|
| Bodies, coats, cushions, plants | Instrument edges, evidence frames |
| Soft dominates emotion | Hard appears in service of clarity |

### Rounded vs angular

Rounded = welcome.  
Angular = focus, never aggression.  
**Forbid:** spikes, chevrons-as-threat, jagged irregular “edgy” silhouettes.

### Forbidden shapes

Perfect toy sphere stacks · military geometry · chaotic bolt clouds · floating neon shards · uncanny hyper-detailed pores as style.

## 3. Color language

### Primary palette (locked)

| Token | Hex | Role |
|-------|-----|------|
| dawn_white | `#F7F4EF` | Air, walls, coats, hope base |
| warm_sand | `#E8DFD2` | Wood, skin warmth, comfort |
| generational_teal | `#2F9EBC` | Brand, UI, medical trust accent |
| midnight_navy | `#1B2A4A` | Depth, text, underlayers |

### Accent palette

| Token | Hex | Role |
|-------|-----|------|
| window_gold | `#F2D4A5` | Hope light |
| soft_sky | `#A9D4E8` | Curiosity air / holograms |
| leaf_green | `#4F8F5B` | Life, growth |
| ember_coral | `#E07A5F` | Urgency **only** |

### Sub-palettes

**Medical:** dawn_white + teal + navy · coral never as décor.  
**Technology:** teal + soft_sky + navy on rounded rects · no cyberpunk magenta.  
**Nature:** leaf_green + warm_sand + window_gold · plants always present when hope is taught.  

### Emotional color scripting

| Beat | Dominant | Support |
|------|----------|---------|
| Curiosity | soft_sky + teal | warm_sand |
| Discovery | teal + window_gold | dawn_white |
| Hope | window_gold + leaf_green | dawn_white |
| Compassion / comfort | warm_sand + dawn_white | gentle teal |
| Intelligence / precision | teal + dawn_white | navy lines |
| Reflection | navy soft + warm_sand | muted teal |
| Wonder | window_gold rim + soft_sky | quiet teal |
| Achievement | window_gold + teal | leaf_green |
| Failure (honest) | warm_sand desat + navy | soft teal return path |
| Recovery / growth | leaf_green rising + gold | dawn_white |

### Lighting philosophy

Signature: **warm key · soft fill · controlled teal rim · honest reflections.**  
Morning discovery is default. Emotion drives exposure.  
Forbid fluorescent-only dread, pure white surgical blowouts, RGB club lighting.

## 4. Materials

| Material | Generational truth |
|----------|-------------------|
| Fabric | Soft matte; coats and curtains carry hospitality |
| Ceramic | Soft satin shells; intelligent, not toy plastic |
| Glass | Clear for evidence & windows; hopeful transmission |
| Metal | Satin small accents only; never chrome-ball joints as style |
| Wood | Warm sand family; domestic and lab furniture honesty |
| Paper / books | Lived intelligence; research presence |
| Plants | Mandatory hope markers in interior teaching spaces |
| Skin | Warm undertone; stylized appeal, not photoreal flesh |
| Hair | Soft graphic masses when human; never greasy uncanny |
| Robot materials | Ceramic-composite + fabric garments; hope, not armor |

### Aging & story

Micro wear OK (softening edges, gentle fabric use).  
Dirt-for-grit, battle damage, apocalypse rust: **forbidden**.  
Materials age like **cared-for instruments**, not abandoned machines.

## 9. Identity rules — NEVER BREAK

### Instant Generational (must)

1. Dawn-forward light or dusk gold — never only grey fluorescent  
2. Teal used as precious accent, not paint flood  
3. Soft shape welcome + clean intelligence edges  
4. Lived-in learning objects (books, plants, tools of care)  
5. Emotional readability without audio  
6. Negative space to breathe  

### Instant breakers (ban)

1. Pure empty grey rooms  
2. Neon cyberpunk / horror clinical  
3. Toy sphere-joint mannequin language as “hero style”  
4. Uncanny MetaHuman skins as brand default  
5. Aggressive military geometry  
6. Clown slapstick color chaos  
7. Logo spam in every corner  
8. Sci-fi menace lighting (blood red / strobe / void black as tone)

### The pause test

Mute. Pause. Ask:

> Would a stranger know this is Generational?

If only the watermark says so, the frame failed the constitution.
