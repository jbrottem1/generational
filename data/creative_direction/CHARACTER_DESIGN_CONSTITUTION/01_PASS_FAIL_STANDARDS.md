# Pass / Fail Standards Checklist  
## Generational Character Design Constitution

**Scoring:** PASS / FAIL / N/A  
**Hero host lock requires:** All **Required** standards PASS. Preferred may be deferred with board note.

---

## Quick gate (hosts)

A permanent Generational host may lock only if:

1. SIL-01, SIL-02, SIL-03 PASS  
2. E-01–E-03 PASS · M-01–M-02 PASS · F-01–F-04 PASS  
3. MEM-01, MEM-02 PASS · I-01–I-03 PASS  
4. T-01–T-04 PASS · A-01–A-03 PASS  
5. TH-01–TH-03 PASS · BL-01–BL-02 PASS  
6. Y-01–Y-04 PASS  

Any FAIL on the quick gate = **NOT READY TO LOCK**.
