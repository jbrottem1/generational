# Generational Animation Style Language

**Owner:** Animation Director (Agent 16) · Veto: Creative Director (Agent 18)  
**Status:** v1 foundation  
**Goal:** Timeless, recognizable animation — not realism.

---

## Signature identity

| Pillar | Rule |
|---|---|
| Silhouette | Readable at Shorts scale in one glance |
| Face | Expressive brows + eyes carry emotion; mouths simple |
| Pose | Every pose reads intent (point, think, celebrate, panic) |
| Color | Bold, limited palettes per series; high contrast subjects |
| Light | Strong key + practical glows; never flat gray sludge |
| Motion | Smooth ease-in-out; slight overshoot on accents |
| Composition | Cinematic 9:16; subject clear in safe zones |
| Proportion | Locked per character version; no episode redesigns |
| Reuse | Prefer library cycles over one-off motion |

---

## Do

- Clean outlines / graphic clarity  
- Kinetic teaching (host interacts with science)  
- Soft gradient / living backgrounds  
- Beat changes every 2–4 seconds  
- Consistent character turnarounds  

## Don't

- Copy any studio, creator, or copyrighted character language  
- Photoreal humans as default host  
- Static talking-head holds  
- Flat solid full-frame backgrounds  
- Random style lottery per episode  
- Clutter that kills educational clarity
