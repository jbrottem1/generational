## Episode factory

1. Topic + sources  
2. Script for motion (Agent 3)  
3. Storyboard package (Agent 16 / automation)  
4. Animation direction (library IDs)  
5. Generate / assemble / Animation QC  
6. Register new reusable assets back to library  

New episodes must feel like the **same show**, not a new art project.
