# Cinematic Camera Language

**Owner:** Animation Director (Agent 16) · Support: Agents 4, 18  
**Rule:** Never leave the camera static unless intentional for storytelling (max 1.5s hold).

---

## Supported verbs

| Preset ID | Shot | Use |
|---|---|---|
| `tracking` | Tracking shot | Follow walk/run through environment |
| `push_in` | Push-in | Hook, reveal detail, emotion |
| `pull_out` | Pull-out | Payoff, scale, context |
| `orbit` | Orbit | Object/character showcase |
| `crane` | Crane | Vertical discovery / lab / canyon |
| `parallax` | Parallax move | Depth while teaching |
| `handheld_energy` | Handheld simulation | Chase, comedy panic (subtle) |
| `establishing_wide` | Wide establishing | New world entry |
| `close_up` | Close-up | Eyes, reaction, fact card |
| `macro` | Macro | Cells, sparks, surface detail |
| `overhead` | Overhead | Diagrams, maps, process |
| `reveal` | Reveal | Door open, light on, curtain |

Machine presets: `library/camera_presets.json`

---

## Composition rules

- Vertical 9:16 safe margins for captions  
- Subject placement follows storyboard emotion  
- Pair camera verb with character action (never both frozen)  
- Prefer motivated camera (follows Dash / object of interest)  

---

## Forbidden

- Locked camera + locked character >3s  
- Random whip pans every cut  
- Zoom spam without story purpose
