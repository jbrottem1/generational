# Failed / Rejected Patterns

Do not reuse without explicit redesign.

| Pattern | Why it failed | Replace with |
|---|---|---|
| Constant walk + arm sway | Distracts from learning | Purposeful choreography beats |
| Ken Burns-only slideshow | Fails Animation QC / feels static | Layered / educator motion path |
| Multi-topic cram in one Short | Confusing; low retention | One concept per episode |
| Rambling filler narration | Pacing drag | Cut any sentence that doesn't teach |
| Overwrite master MP4 | Loses A/B history | `unique_path` / `_vN` |
| Wave gesture loops | Looks amateur | Idle / point / push only when needed |
| Lecture without on-screen demo | "Tell" not "show" | Demo overlay synced to words |
| Blind worktree merge (15/16) | Architecture risk | Agent 1 merge plan |
