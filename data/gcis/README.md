# GCIS Data Root

Permanent continuous-improvement memory for Generational.

| Path | Purpose |
|---|---|
| `dashboard.json` | Live company metrics |
| `department_review.json` | Department operational review |
| `agent_performance.json` | Specialist agent cards |
| `knowledge/` | Lessons, standards, reusable assets, prompts |
| `templates/` | Post-production + sprint report templates |
| `reviews/` | Completed reviews & executive reports |

Charter: `/GCIS.md`  
Refresh dashboard: `./venv/bin/python scripts/gcis_refresh_dashboard.py`
