# Phase VIII — Before / After Rationale

Each sculpt change must earn its place.

---

## Rank 1 — Coat as logo silhouette

**Before:** Coat read as ribbed egg / weak cylinder; blackout failed.  
**After:** Wider body, stronger hem flare, deeper lapels, stand collar, soft shoulder overhang, longer sleeve language.

| Lens | Effect |
|------|--------|
| Character | He becomes a *medical mentor*, not a nude robot wearing paint |
| Memorability | Coat is the drawable logo |
| Trust | Parents/teachers read “educator / clinician” |
| Silhouette | Primary blackout rescue |
| Animation | Coat bones gain readable hem/follow-through mass |

---

## Rank 2 — Egg head + integrated cranial crescent

**Before:** Ball + beanie = white robot.  
**After:** Non-uniform egg; navy crown flattened and sunk into skull.

| Lens | Effect |
|------|--------|
| Character | Unique cranial identity |
| Memorability | Profile drawable without color |
| Trust | Integrated tech feels designed, not cosplay |
| Silhouette | Crescent break in head outline |
| Animation | Same head bone; facial keys still upper-face |

---

## Rank 3 — Eye trust hierarchy

**Before:** Glow / doll risk.  
**After:** Larger sclera, prouder −Y, darker pupil, quieter emission, thicker lids.

| Lens | Effect |
|------|--------|

| Memorability | Teal iris + dark pupil child can recall |
| Trust | Immediate “safe to learn” |
| Silhouette | Secondary (front read) |
| Animation | Blink/squint/gaze more readable |
