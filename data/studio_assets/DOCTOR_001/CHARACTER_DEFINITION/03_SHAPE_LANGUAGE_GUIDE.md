# DOCTOR_001 — Shape Language Guide

---

## Grammar

| Word | Shape | Used for |
|------|-------|----------|
| Soft oval | Trust | Cheeks, coat hem, soft shoulder |
| Gentle egg | Head appeal | Cranium (elongated soft, not ball) |
| Rounded rectangle | Intelligent tech | Cranial panel inset, pin, UI props |
| Soft arc | Discovery | Brow, smile line, coat lapel curve |
| Clean vertical | Calm professionalism | Coat front edges, posture axis |
| Subtle crease | Life | Smile folds, lid line — never scars |

## Forbidden shapes

| Shape | Why banned |
|-------|------------|
| Perfect sphere stacks | Toy / greybox identity |
| Aggressive spikes / wedges | Threat |
| Hard hex armor | Military |
| Random bolt clouds | Clutter; not elegant android |
| Flat floating panels | Fail costume and silhouette |

## Proportion targets (Concept C)

- Height: **1.85 m** (6'1") — keep pipeline scale  
- Head scale: **~1.08** vs human realist (appeal)  
- Shoulder width: approachable heroic — not bodybuilder  
- Waist under coat: readable taper  
- Arms: anatomically plausible; hands slightly larger for CU teaching  
- Legs: long enough for elegant walk; not stilts  

## Design mantra

**Curves for heart. Verticals for mind. Coat volumes for brand.**
