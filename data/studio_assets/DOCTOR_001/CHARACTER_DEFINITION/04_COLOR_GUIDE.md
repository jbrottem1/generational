# DOCTOR_001 — Color Guide

Aligned to Generational Style Constitution v3.

## Core palette

| Token | Hex | Role on Doctor |
|-------|-----|----------------|
| dawn_white | `#F7F4EF` | Coat primary, shell ceramic |
| warm_sand | `#E8DFD2` | Face / gentle glaze |
| midnight_navy | `#1B2A4A` | Cranial crescent, underlayer, soft joint fairings |
| generational_teal | `#2F9EBC` | Iris warmth, coat piping, insignia |
| window_gold | `#F2D4A5` | Rare hope specular / dawn rims only |
| soft_sky | `#A9D4E8` | Environment support — not on body |

## Hierarchy

1. Dawn white dominates (coat + shell)  
2. Navy defines iconic cranial + under accents  
3. Teal is **rare and precious** (pin, thin piping, iris)  
4. Gold never paints the body — lighting only  

## Emotional scripts (Doctor shots)

| Beat | Palette emphasis |
|------|------------------|
| Welcome | dawn_white + warm_sand + soft teal iris |
| Discovery | teal accent + window gold light |
| Comfort | warm_sand face + dawn coat |
| Precision | navy cranial + teal piped edge |
| Hope | gold rim light behind Doctor |

## Banned color behaviors

- Chrome black elbows/knees  
- Neon eye glow  
- Random RGB emissives  
- Dirty greys that kill hope  

## Coat piping rule

Teal exists as a **line**, not a panel. Children should remember “teal trim,” not “blue rectangles stuck on sides.”
