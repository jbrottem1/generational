# DOCTOR_001 — Facial Guide

## North star

Kindness is visible **before** dialogue.

---

## Overall proportions (Concept C)

- Soft egg cranium; integrated navy cranial crescent  
- Slightly elevated brows (intelligent, not stern)  
- Mid-face warmth (cheeks soft, never balloon spheres)  
- Clear mouth plane with lips that can smile **and** speak  
- Friendly jaw; no goatee-cone spike  
- Smooth neck into collar  

## Eyes (emotional anchor)

| Element | Spec |
|---------|------|
| Scale | Large enough for CU trust; not doll-stare |
| Iris | Warm teal-blue; readable ring |
| Pupil | Dark, decisive; hierarchy over iris |
| Sclera | Soft white; not plastic glare |
| Lids | Real upper/lower volume; blinks close convincingly |
| Highlight | Small offset specular — alive, not flashlight |
| Emission | Near-zero; trust over tech glow |
| Gaze | Converges on learner / evidence; asymmetric micro ok |
| Rest | Soft squint + warmth |

## Mouth

- Separate upper/lower lip volumes  
- Smile widens corners up; does not delete lip shape  
- Visemes change aperture shape, not only jaw  
- Resting soft smile always present at low value  

## Brow

- Split L/R capable of asymmetry  
- Curiosity = outer or inner raise depending on beat  
- Concern = inner raise without anger fold  

## Expression dictionary (silent-readable)

| State | Face |
|-------|------|
| Neutral welcome | Soft smile, open eyes, mild squint |
| Curiosity | Brow raise, slight head tilt, lips soft |
| Teaching | Focused eyes, gentle smile, mouth mid-open optional |
| Empathy | Softened lids, soft smile, forward attention |
| Concern | Inner brow, settled mouth, caring eyes |
| Joy | Fuller smile, cheek lift, brighter eye squint |
| Determination | Lowered brow slightly, clear direct gaze |
| Thinking | Asymmetric brow, eyes shift, mouth closed soft |

## Micro-expression rule

Never perfect symmetry at rest. 2–5% asymmetry = life.

## Forbidden face language

- Blank sphere  
- Bar mouth  
- Headlight eyes  
- Balloon cheek orbs  
- Painted-on features
