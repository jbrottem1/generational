# DOCTOR_001 — Character Design Manifesto

**Phase:** VII · Character Definition Studio  
**Status:** Design specification — **not yet approved for Evolution implementation**  
**Architecture:** Frozen · Identity only  

---

## The problem we are solving

The current Doctor is a technically capable procedural actor.

He is not yet an icon.

If a child draws him from memory tomorrow, they will draw **a white robot with a blue hat** — not **The Doctor**.

That fails Generational.

---

## The purpose of this studio

Stop polishing primitives.

Define the permanent visual identity first.

Only after approval does Character Evolution resume — one category at a time — against this Bible, not against the greybox habit.

---

## Principles (non-negotiable)

1. **Trust before tech.** Warmth and kindness must read before any mechanical cleverness.
2. **Appeal over photorealism.** Generational is iconic educational cinema, not MetaHuman.
3. **Silhouette is law.** If shadow fails, redesign.
4. **One glance, one truth.** Eyes create trust without dialogue.
5. **Coat is brand.** Clothing alone must say medical educator.
6. **Memorable, not generic.** Stand beside Wall-E / Baymax / Iron Giant in originality of feeling — never copy their shapes.
7. **Runtime sacred; appearance not.** Bone names and pipelines stay. Everything visual may change to serve identity.
8. **Educational presence.** Parents and teachers should feel safe leaving a child with this host.
