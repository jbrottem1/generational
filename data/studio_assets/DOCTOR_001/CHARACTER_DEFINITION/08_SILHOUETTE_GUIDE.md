# DOCTOR_001 — Silhouette Guide

## Pass / fail

| Test | Requirement |
|------|-------------|
| 20 m | Recognizable as The Doctor |
| Shadow only | Same |
| No color | Same |
| No face detail | Coat + cranial crescent + proportions still read |

**Current prototype: FAIL** (sphere-egg-robot).  
**Target Concept C: PASS** required before v1.0.

---

## Silhouette ingredients (must-have)

1. Tall coat rectangle with **hem flare**  
2. Soft shoulder masses  
3. Cranial **crescent / dome** profile (not bare ball)  
4. Slight head overscale for appeal  
5. Optional signature: open-palm teaching arm  

## Silhouette killers (ban)

- Floating limb gaps  
- Egg torso without coat edges  
- Perfect sphere head  
- Chrome joint balls as silhouette dots  
- Teal slab “wings”  

## Blackout drawing exercise

Designers must be able to draw the silhouette in 10 seconds:

> Coat flare · crescent head · soft shoulders · one open hand

If the drawing needs color to explain “doctor,” redesign.

---

## Thumbnail / mural

Silhouette must work at:

- App icon simplified mark  
- End-card  
- Outdoor banner  

Pin may become a simplified diamond/teal mark in icon reductions.
