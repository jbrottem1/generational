# The Generational Method

**Status:** ACTIVE teaching doctrine  
**Owner:** Agent 0 (PMO) · Animation Director: Agent 16 · Creative veto: Agent 18  
**Character:** CHAR-STICK-001 — Generational Educator  

We optimize for **intentional teaching**, not for more movement.

---

## Core law

> Every movement must have a purpose.  
> Every word must have value.  
> Every second must increase understanding.

If a gesture, walk, pause, or sentence does not serve the lesson — cut it.

---

## Reject

- Random waving  
- Constant arm movement / idle fidgeting  
- Walking without purpose  
- Looking around randomly  
- Repeating gestures  
- Long pauses / rambling / filler  
- Static lectures / monotone delivery  
- Motion for motion’s sake  

---

## Lesson structure (every Short)

1. **Hook** (2–3s) — curiosity spike  
2. **Demonstration** — show the concept happening  
3. **Explanation** — name the idea precisely  
4. **Real-world example** — everyday intuition  
5. **Key takeaway** — one memorable line  
6. **Curiosity bridge** — want the next lesson  

---

## Movement rules

| Allowed | When |
|---|---|
| Idle / calm stance | Default — confident, grounded, facing audience |
| Think | Brief beat before a key reveal (≤1s) |
| Point | Indicating the object/event being named |
| Push / pick / drop | Physical demonstration of force/motion |
| Present | Opening arms to a board/result (sparingly) |
| React | Only when the experiment surprises |
| Walk | Only to move between demo stations |

**Default state is stillness with life** (subtle weight shift + blink) — not choreography spam.

---

## Show, don’t just tell

Prefer demonstration over definition. Sync visuals to words:

- “This ball…” → already indicating/holding it  
- “Notice what happens…” → event occurs immediately  

---

## Camera

Cinematic motion only when it clarifies. Prefer stable framing on the lesson.

---

## Review checklist (must pass before export)

| Category | Pass if |
|---|---|
| Teaching clarity | One core concept lands |
| Animation quality | Smooth, grounded, non-robotic |
| Gesture quality | Purposeful; no fidget spam |
| Synchronization | Words match on-screen action |
| Engagement | Hook + pace; no drag |
| Scientific accuracy | Correct; not misleading |
| Visual storytelling | Concept is *seen* |
| Retention potential | Memorable takeaway |

---

## Audience test

Viewer should think: “I didn’t know that.” / “Now I understand.” / “Show me the next one.”  
Never: “This is dragging.”
