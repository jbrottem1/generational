# Animation Engine V1 → superseded by V2

See **[ANIMATION_ENGINE_V2.md](./ANIMATION_ENGINE_V2.md)** for the current cinematic quality layer.

V1 introduced motion minimums (every scene must move).  
V2 keeps the same frozen pipeline position and elevates planning to **motivated cinematography + immersion tests**.

Legacy CLI `scripts/animation_engine_v1.py` still builds packages through the shared `services/animation_engine` module (now version `2.0.0`). Prefer `scripts/animation_engine_v2.py`.
