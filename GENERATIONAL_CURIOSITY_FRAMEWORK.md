# Generational Curiosity Framework

**Status:** LOCKED · Company standard  
**Owner:** Agent 0 (executive) · maintained by Agent 27  
**Effective:** 2026-07-11  
**Applies to:** All educational Shorts

---

## Forbidden openings

Do **not** begin lessons with:

- "Today we're going to learn…"
- "Welcome back…"
- "In this video…"
- "Hey guys…" / generic greetings

---

## Required structure

| Time | Beat | Intent |
|------|------|--------|
| 0–3s | **Curiosity question** | Unanswered; viewer thinks "I need to know." |
| 3–6s | **Surprising accurate statement** | Scientific, not hype |
| 6–10s | **Promise** | What they will understand |
| Mid | **Lesson + visual demo** | One concept; board / diagrams |
| Mid | **Real-world connection** | Concrete example |
| End | **Memorable takeaway** | One sentence |
| End | **Curiosity bridge** | Tease another fascinating topic |

---

## Pacing

Fast. Confident. Energetic.  
Every sentence adds new information.  
No filler. No repetitive transitions.

## Science

Claims must match reliable sources (peer-reviewed, university, museum, textbooks, consensus).  
State uncertainty when understanding is evolving.  
Never exaggerate for drama.

## Studio (current)

White background · Professor Gen · Whiteboard · Simple diagrams · Color highlights · Minimal distraction.
