# Project Fluid Motion

**Status:** ACTIVE — animation quality freeze  
**Owner:** Agent 0  
**Law:** Better movement, not more movement. Believable performance over AI motion.

## Freeze

No new environments · props · effects · features · departments.  
Every change must improve **animation quality**.

## Principles

Natural balance · weight · momentum · anticipation · follow-through  
Ease in / ease out · smooth acceleration / deceleration  
Natural posture · proper timing · intentional movement  
Stillness is valid when it strengthens communication.

## Body performance

Breathing · eye drift · blink timing · subtle head · weight shifts  
Balanced stance · purposeful walk / turn · meaningful gestures · strong silhouettes  
Reject: constant waving, fidgeting, floating, sliding, snaps, teleports, purposeless motion.

## Gesture system

Every gesture communicates. Point to emphasize. Open hands to invite. Lean on reveal.  
Pause before important beats. Default = calm idle.

## Engine

`services/animation/fluid_motion.py` — easing, pose blend, anticipation, follow-through, mouth EMA, life signals.  
Wired through `performer.py` + `stick_figure.py` for all educator performances.
